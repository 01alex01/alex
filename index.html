    Stylish Cup with Coffee

    I created a visually appealing cup with coffee using CSS. This web design project allowed me to showcase my CSS skills and attention to detail. The cup is meticulously crafted with a clean and modern design, featuring a grey background and a circular shape achieved through the clever use of CSS properties. To bring the cup to life, I utilized CSS positioning techniques to position the coffee inside the cup. The coffee is represented by a rich, inviting color that adds depth and realism to the overall design. By carefully adjusting the size and positioning, I was able to create a visually pleasing representation of a cup filled with hot coffe. For added flair, I implemented an optional steam effect using CSS animations. The steam gracefully rises from the cup, giving the design a touch of dynamism and creating a sense of warmth and freshness. This project allowed me to demonstrate my proficiency in CSS styling, including the use of properties like background colors, borders, positioning, and animations.

    Shotlet Apartment

    Experience modern urban living with the interactive virtual tour of the Shotlet Apartment. As a web developer, I designed this immersive web project to showcase my skills in web development and user experience design.
    The Shotlet Apartment offers a visually stunning and interactive virtual space for users to explore. Through HTML, CSS, and JavaScript, I created a seamless and responsive interface that allows visitors to navigate different rooms and experience the apartment's layout and design.
    With meticulous attention to detail, the virtual tour captures the essence of contemporary living. The sleek furniture, carefully chosen color palette, and stylish animations create an enticing and immersive experience.
    Interactive elements enhance user engagement, allowing visitors to click on objects for additional information about the apartment's features. This interactive approach adds depth and interactivity to the virtual tour.
    The Shotlet Apartment is a testament to my expertise in web development and my ability to deliver captivating user experiences. It showcases the possibilities of virtual tours in the real estate industry, making it a valuable addition to my portfolio as a web developer.
    By including the Shotlet Apartment, I aim to demonstrate my skills in web development and highlight my commitment to creating exceptional online experiences.
